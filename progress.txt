## Codebase Patterns
- Node em: /c/Users/User/nodejs/node-v20.11.1-win-x64/node.exe — usar PATH="..." prefix
- PATH prefix obrigatório: PATH="/c/Users/User/nodejs/node-v20.11.1-win-x64:$PATH"
- App fica em: crm-alliance/ (subdiretório do projeto)
- Commits seguem: feat: [Story ID] - [Story Title]
- TypeScript estrito — sem any, sem as unknown
- Server Components por padrão; 'use client' só quando necessário (hooks, Framer Motion, dnd-kit)
- Animações: importar de @/lib/animations.ts (pageTransition, modalAnimation, staggerContainer, etc)
- Cores via tokens Tailwind: alliance-blue (#1E90FF), alliance-dark (#0A2EAD), alliance-mid (#1565C0)
- Supabase client: @/lib/supabase/client.ts (browser) e @/lib/supabase/server.ts (server)
- Tipos do banco: @/lib/supabase/types.ts — nunca importar direto de @supabase/supabase-js
- BlobBottom e BlobHeader já implementados em @/components/layout/
- NavShell, AllianceBadge, AllianceCard, PageTransition já implementados
- shadcn/ui components disponíveis em @/components/ui/
- Formato de data: date-fns com locale ptBR
